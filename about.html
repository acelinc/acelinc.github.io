嗨~我是穆鑫，UI 设计师一名，坐标魔都上海。

16 年底从机械自学转行 UI 以来，已在设计这行耕耘 3 年多，独立负责过 3 款 C 端移动端产品从 0 到 1 的 UI 设计，也跟团队协作设计过 B 端产品。

对比下来，还是对 C 端产品更感兴趣，不仅能用设计帮公司解决商业上的问题，也能近距离地解决用户的问题，给用户带去生活上的便利。

目前负责一款健康健美类的产品「生酮减肥」，从 0 用户做到现在每天稳定的新增用户，期间通过一次设计改版大幅度提高了产品的分享率和好评率。

虽然做设计不能改变世界，虽然手机是冰冷的，但依然希望可以通过设计这项技能，给公司带去商业上和用户上的增长，同时给用户做出一些带有「温度」的产品。

  • 喜欢音乐，Miku 赛高；
  • 骑车入门爱好者，一辆坦克达，每天 16 公里风雨无阻。
